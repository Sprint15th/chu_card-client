## 📌 Background

친구가 생일인데, 가볍게 ‘생일 축하해’라고 보내기보단 색다르게 보내고 싶을 때 있지 않았나요?

생일을 축하할 때 축하 메시지를 색다르게 보낼 수 있는 서비스입니다!

**Target**

- 지인이 생일일 때 식상한 메세지보다 **특별한 선물**을 주고 싶은 사람
